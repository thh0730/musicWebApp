### 基于Vue全家桶仿制的网易云音乐H5页面

 **所用技术栈：**
    Vue.js Vuex Vue-Router Axios 等

 **运行**
1. 克隆项目后先下载运行依赖

```
npm install
```
2. 下载网易云音乐服务并启动

```
https://github.com/Binaryify/NeteaseCloudMusicApi
```
3. 运行项目

```
npm run serve
```

 **数据来源：** 
网易云音乐API：https://binaryify.github.io/NeteaseCloudMusicApi/#/
